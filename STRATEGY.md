## 動能評分公式

每檔 ATH 股獨立計分（0~100，三層加權）：

### 🥇 Tier 1（核心，3 選 1 給 25 分）

| 訊號 | 條件 | 統計續漲率 |
|---|---|---|
| **漲停鎖死** | 漲幅 ≥ 9.5% AND 量比 < 1.2x | 88-93% |
| **量爆價揚** | 量比 ≥ 3 AND 漲幅 ≥ 5% | 78-85% |
| **跳空缺口** | 開盤 > 昨高 × 1.005 AND 收 > 開 | 75-82% |

### 🥈 Tier 2（各 +10~15 分）

| 訊號 | 條件 |
|---|---|
| RSI 健康強勢區 | 60 ≤ RSI ≤ 75 |
| 多頭排列加速 | close > MA5 > MA10 > MA20 |
| 創 ATH | today >= 2y_max × 0.999 |
| 收盤近高 | close ≥ (high - range × 0.2) |
| 長紅 K | (close-open) / range ≥ 0.7 |
| 族群同步 | 同族群 ≥60% 今日漲 |

### 🥉 Tier 3（V3 美股加分，+5~10）

| 訊號 | 加分 | 對應 TW 族群 |
|---|---|---|
| QQQ 昨日 +0% 以上 | +5 | 全 7 個科技族群 |
| SMH 昨日 +0.5% 以上 | +10 | 半導體、其他電子 |
| IGV 昨日 +0.5% 以上 | +10 | 通信網路、電腦及週邊 |

### 評級

| 分數 | 評級 | 隔日續漲率 |
|---|---|---|
| ≥ 80 | ⭐⭐⭐ | ≥ 85% |
| 60-79 | ⭐⭐ | 70-85% |
| < 60 | ⭐ | < 70% |

## 出場規則

### 雙重 Trailing Stop

```
持倉中每日檢查
    │
    ▼ 條件 1：今日收盤 < 20MA → 隔日開盤賣出
    ▼ 條件 2：當前價 < 進場後峰值 × 0.7（-30%）→ 立刻賣出
    │
    （二者擇一觸發）
```

**統計**：5 年回測中 100% 出場都是 20MA 觸發，-30% 從未啟用 → 20MA 已經是足夠的 trailing stop。

**程式**：`tw_breakout_filter.analyze()` 內的 `is_exit` 欄位

## 📞 改策略時的決策流程

1. **看每日 LINE 盤後檢討**
2. **若週勝率 < 45%**：暫停 1 週、提高動能門檻、檢查大盤體制
3. **若有族群連 3 天虧損**：考慮從 ALLOWED_INDUSTRIES 移除
4. **若有訊號連 5 次失敗**：降低該訊號 Tier 權重
5. **重大改動**：先用 `backtest_v3.py` 套用新規則回測 → 對比 PF/CAGR → 決定是否上線
